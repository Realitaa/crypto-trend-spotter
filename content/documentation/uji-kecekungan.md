# Uji Kecekungan (Turunan Kedua)

Rumus beda hingga turunan kedua:

<Katex expr="f''(x) \approx \frac{f(x+h) - 2f(x) + f(x-h)}{h^2}" display />

### Interpretasi

- <Katex expr="f''(x) > 0" /> → grafik **cekung ke atas** → **uptrend semakin kuat**
- <Katex expr="f''(x) < 0" /> → grafik **cekung ke bawah** → **downtrend semakin kuat**
- <Katex expr="f''(x) \approx 0" /> → perubahan tren (potensi reversal)
