# Tentang Crypto Trend Spotter

Crypto Trend Spotter adalah website untuk menganalisis pergerakan harga kripto dan menghasilkan wawasan yang dapat lebih mudah dipahami pengguna. Latar belakang dari proyek ini adalah untuk menggunakan pendekatan matematis objektif ketimbang perasaan saat melakukan transaksi mata uang kripto yang memiliki volatilitas tinggi. Trader juga sering terlambat menyadari perubahan tren (membeli ketika di pucuk atau menjual di lembah). <Rated />

## Deskripsi Singkat

Crypto Trend Spotter menganalisis pergerakan harga kripto secara matematis untuk mengetahui apakah pasar sedang naik, turun, atau berpotensi berbalik arah.  
Semua dihitung otomatis menggunakan _turunan numerik_ dan _pola kecekungan_ agar pengguna mendapatkan insight yang lebih presisi tanpa menghitung manual.

Aplikasi ini dibangun sebagai tugas mata kuliah _Kalkulus Diferensial_ dan berfokus pada penerapan konsep turunan ke dalam pemrograman modern, termasuk visualisasi data dan pengolahan sinyal sederhana.

---

## Tujuan Proyek

- Menerapkan konsep turunan numerik untuk menganalisis perubahan harga.
- Menguji kecekungan (concavity) untuk mendeteksi potensi reversal tren.
- Mengintegrasikan API publik untuk memperoleh data harga kripto.
- Membangun antarmuka visual dan interaktif menggunakan Nuxt.
- Mendokumentasikan hasil riset dan implementasi teknis.

---

## Teknologi yang Digunakan

- **Framework:** Nuxt 4
- **Bahasa:** TypeScript
- **Styling:** Tailwind CSS 4
- **UI Library:** Nuxt UI 4
- **Data Source:** Binance API
- **Charting:** Lightweight Chart
- **Bundler:** Bun
- **Documentation:** Nuxt Content

## Fitur Utama

- Mengambil dan menampilkan data harga kripto secara real-time.
- Menghitung turunan numerik untuk mendeteksi kenaikan/penurunan.
- Menghitung kecekungan untuk memprediksi potensi pembalikan tren.
- Menampilkan grafik harga dan indikator tren.
- Sistem pencarian aset kripto.
- Dokumentasi otomatis berbasis Markdown melalui Nuxt Content.

## Kesimpulan

Crypto Trend Spotter membuktikan penerapan konsep kalkulus — khususnya diferensiasi numerik dan analisis kecekungan — dapat memberikan insight nyata dalam dunia analisis pasar kripto.  
Proyek ini tidak hanya berfungsi sebagai aplikasi, tetapi juga sebagai demonstrasi bagaimana teori matematika dapat diimplementasikan dalam perangkat lunak modern secara efektif.
